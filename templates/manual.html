How to Use Sign-nificant?

In order to deploy this app in your local host, you will be required to download the source code first which is availabe on Github

The instruction step-by-step were also included there :D

Steps:

  • Step 1: Clone Repository

    • git clone https://github.com/yasminhannahza/sign-nificant.git

  • Step 2: Create virtual environment (venv) using conda

    • conda create -n VENV_NAME_HERE

  • Step 3: Activate virtual environment (venv) using conda

    • conda activate VENV_NAME_HERE

  • Step 4: Install dependencies in virtual environment (venv)

    • pip install -r requirements.txt

  • Step 5: Change directory to project directory

    • cd (PATH/TO/sign-nificant)

  • Step 6: Run application on Local Host

    • python app.py

    or

    flask run
