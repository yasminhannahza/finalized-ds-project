About Sign-nificant

This is my Data Science Project where,

This project aims to:

  • To be able to identify the specific 5 sign language made by hand gestures of the end users through computer vision
  • To train a model using LSTM algorithm to recognize 5 dynamic sign language with the highest possible accuracy.
  • To present an interactive way the meaning of the 5 dynamic sign language to the users using suitable application framework

Functionality:

  • Able to attract people to learn sign language
  • Help to bridge the communication gap between hearing and non-hearing individuals
  • Can be used as an educational tool for people who are learning sign language, allowing them to practice and improve their skills

Potential stakeholders:

  • School
  • Families and caregivers of individuals who use sign language
  • Interpreters and translators for sign language

This project is open-source and the code is available on Github
